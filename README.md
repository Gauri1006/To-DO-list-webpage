#  To-Do List Web App

A beautiful and interactive **To-Do List Website** built using **HTML, CSS, and JavaScript** with a pastel UI 🎀✨

---

## 🚀 Features

* ✅ Add new tasks
* ✏️ Edit existing tasks
* ❌ Delete tasks
* ✔️ Mark tasks as completed
* 🔍 Filter tasks (All / Completed / Pending)
* 💾 Saves tasks using Local Storage
* 🌙 Dark Mode toggle
* 📊 Task completion counter
